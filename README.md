# Vanilla JS CRUD + Laravel REST API

Полный цикл CRUD без JS-фреймворков. Fetch API, валидация, адаптивная верстка.

## Стек
Laravel, SQLite, Blade, Tailwind CSS, Vanilla JS, REST API

## Возможности
- Динамический интерфейс без перезагрузки страницы
- Валидация на клиенте и сервере
- Адаптивная вёрстка на Tailwind
- Чистая архитектура Laravel

## Запуск
1. `composer install`
2. `php artisan migrate`
3. `php artisan serve`
4. Открой `http://vanilla-crud-dashboard.test`
